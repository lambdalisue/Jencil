Jencil is a JavaScript cross-browser WYSIWYM and WYSIWYG editor which is working as jQuery plugin and written in CoffeeScript. It allows you to turn any textarea into a WYSIWYM (markup) and WYSIWYG (rich content) editor. See Demo to get what Jencil does.

For quick use, see Tutorial to get how to install and use Jencil on your page.

This editor is completely customizable with plugin! See Document for more detail if you are interested in developing Jencil plugin.

Features

  • Free and Open source
  • 100% skinnable using Theme (CSS)
  • 100% customizable using Plugin (JavaScript)
  • Easy to use
  • Easy to understand & extend
  • WYSIWYM (Markup) Editor with Preview is available
  • WYSIWYM Preview template HTML is customizable
  • WYSIWYM Markup Parser is customizable
  • WYSIWYG (Rich) Editor with Raw text editing is available
  • WYSIWYG template HTML (WYSIWYG Editor style) is customizable
  • Multiple document type is available
  • Full screen editing mode is available (not yet)
