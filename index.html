Jencil is a JavaScript cross-browser WYSIWYM and WYSIWYG editor which works as jQuery plugin and written in CoffeeScript. It allows you to turn any textarea into a WYSIWYM and WYSIWYG editor. See Demo to get what Jencil does.

Features

  • Free and Open source
  • 100% skinnable using Theme (CSS)
  • 100% customizable using Plugin (JavaScript)
  • Easy to use
  • Easy to understand & extend
  • WYSIWYM (Markup) Editor with Preview is available
  • WYSIWYM Preview template HTML is customizable
  • WYSIWYM Markup Parser is customizable
  • WYSIWYG (Rich) Editor with Raw text editing is available
  • WYSIWYG template HTML (WYSIWYG Editor style) is customizable
  • Multiple document type and dynamic change
  • Keyboard shortcut (not yet)
  • Full screen editing mode (not yet)
  • File attach system is (not yet)

How to use

  1. Download latest Jencil and extract it
  2. Copy package directory to your javascript directory and rename it to jencil
  3. Add the following line to your HTML Head: 
    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>
<script type="text/javascript" src="js/jencil/jencil.min.js"></script>
<script type="text/javascript">
    $(document).ready(function(){
        $('textarea#document-body-demo').jencil({
            documentTypeElement: $('select#document-type-demo')
        });
    });
</script>
  4. Add the following line to your HTML Body: 
    <select id="document-type-demo">
    <option value="html">HTML</option>
    <option value="wysiwyg">WYSIWYG</option>
</select>
<textarea id="document-body-demo">This is a default text</textarea>
