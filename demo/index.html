Jencil alpha Demo

Cross browser WYSIWYM and WYSIWYG editor written in CoffeeScript

See Textarea if you are interested in munipulating textarea. See Richarea if you are interested in munipulating iframe. Jencil use these library to munipulating textarea and iframe.

Demo

H1 element is shown as Red text in WYSIWYG mode. That because the WYSIWYG mode use RichEditor and RichEditor can set Template HTML. Used template in RichEditor has 'h1{color: red}' setting on its head so H1 element is shown as Red text only in WYSIWYG. That mean you can design your RichEditor as possible as CSS can.

Features

  • Easy to use (working as jQuery plugin)
  • Cross browser (but doesn't support old browser)
  • WYSIWYM and WYSIWYG compatible
  • Multiple markup is supported
  • Profile is supported
  • Theme is supported
  • Plugin is supported
