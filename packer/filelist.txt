ParseMaster.pm   - Perl module (equivalent to ParseMaster.js)
Pack.pm          - Perl module (equivalent to Pack.js)
jsPacker.pl      - Perl commandline program to pack javascript source
jsPacker.exe     - Compiled Win32 version of jsPacker.pl
SamplePacker.pl  - Perl commandline program to generate an html page containing examples of packed Javascript
SamplePacker.exe - Compiled Win32 version of SamplePacker.pl
SamplePacked.htm - Html output generated by SamplePacker.pl
